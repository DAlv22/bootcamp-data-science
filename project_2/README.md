# Instacart Grocery Delivery Analysis

This project focuses on analyzing data from Instacart, a grocery delivery platform, to understand customer behavior and trends. The dataset provided has been modified to reduce size and introduce missing and duplicate values while maintaining the original data distributions.

### Objectives

1. Data Preprocessing: Clean and preprocess the dataset to handle missing and duplicate values, ensuring data quality for analysis.
2. Exploratory Data Analysis (EDA): Analyze the distribution of orders, products, and aisles to gain insights into customer behavior and preferences.
3. Conclusion: Summarize findings and insights from the analysis, highlighting any trends or patterns observed in customer purchasing behavior.
